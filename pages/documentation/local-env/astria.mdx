## Creating Astria Account

To use Astria, you need to create an account on [Astria](https://www.astria.ai/). Once you have created an account, 
you can get your API key from the [API page](https://www.astria.ai/users/edit#api). Afterwards in your .env file, you
need to make the following changes:

import { Steps } from 'nextra/components'
 
<Steps>
### Replacing API Key
 
Replace `your-astria-api-key` with your original `Astria API key` from [API page](https://www.astria.ai/users/edit#api).
 
### Replacing Webhook Secret

Replace `your-app-webhook-secret` with any arbitrary URL friendly string eg.`shadf892yr398hq23h`

</Steps>
